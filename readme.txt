Simple Contact form 7 Wizard Multisteps Form , light weight no ajax simple jquery and easy.

== Description ==

Contact form 7 wizard form and in the end would send an email with all the info collected.  This plug-in does just that.  This plugin requires the Contact Form 7 Word press plugin.

Demo  [Click Here](http://wpsenior.com/simple-contact-form-7-wizard-multisteps-form/)

[vimeo https://vimeo.com/189633455]

**Simple Shortcode**
`Example form

[step s="1" title="step 1 "]

<label>Email (required)[email* your-email] </label>
<label>Name (required)[text* your-name] </label>

[nextbutton]
[/step]

[step s="2" title="step 2"]

<label>Phone (required)[phone* your-phone] </label>

[backbutton] [nextbutton]
[/step]

[step s="3" title="step 3"]

<label>Message (required)[textarea* message] </label>

[backbutton] [submit "Send"]
[/step]`
